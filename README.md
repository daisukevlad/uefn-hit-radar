# UEFNヒットレーダー

「今、UEFNで何を作ればヒットするか」を**Epic公式データ**からリアルタイムに調査し、
**個人開発者(1日3時間・発信力なし)向けの最適解**を日本語で提示するアプリです。

## 使い方(これだけ)

1. **`起動.bat` をダブルクリック**する
2. ブラウザが開くので待つ(初回は3〜5分の自動調査があります)
3. 画面の「**今日の結論 — あなたが作るべきマップ TOP5**」を見る

以上です。毎日1回起動する(または画面右上の「最新データを取得」を押す)と、
日々の変化と急上昇マップの検知精度がどんどん上がります。

## 何をしているのか

- Epic公式の **Fortnite Data API**(認証不要・利用規約上公開されているAPI)から、
  新着マップ約3万本のカタログと、約3,000マップの実測データ
  (同時接続数・プレイ回数・平均プレイ時間・翌日継続率など)を毎日取得します。
- ジャンル(タグ)ごとに **需要(実測プレイヤー数)÷ 競合(新規リリース数)** を計算し、
  さらに「平均プレイ時間 × 継続率 × 個人開発のしやすさ」で重み付けした
  **機会スコア**でランキングします。

### なぜ「厚利少売」設計なのか

フォートナイトの収益(エンゲージメントペイアウト)は **人数×時間** で決まり、
Discover(発見)画面の掲載順は **継続率とプレイ時間** で決まります。
つまり「少数のプレイヤーに深く長く遊ばれるマップ」こそが、
発信力ゼロの個人開発者にとって収益もDiscover露出も最大化する最適戦略です。
このアプリのスコアは、1v1・ボックスファイトのような薄利多売の飽和ジャンルが
自動的に不利になるよう設計されています。

## 画面の見方

| セクション | 意味 |
|---|---|
| 今日の結論 TOP5 | 今作り始めるべきジャンル。参考にすべき成功マップのコード付き |
| 急上昇レーダー | 最近リリースされて既に人が付いている新作(=今動いているトレンド) |
| ジャンル分析 | 全ジャンルのスコア・需要・競合の一覧 |
| 定番トップマップ | 継続的に強いマップの定点観測(使うほど充実) |

## 必要環境

- Windows + Node.js(インストール済みを確認済み)
- インターネット接続
- 外部サービス登録・APIキー・料金は一切不要

## 自動で毎日実行したい場合(任意)

おすすめは**クラウド収集**です。`セットアップ_GitHub連携.bat` を1回ダブルクリックし、
画面の案内に従ってGitHubにログイン(アカウントがなければその場で無料作成)するだけで、
**PCの電源が入っていなくても** GitHubのクラウドが毎朝7:05に自動収集してくれます。
アプリを起動すると、その結果が自動でダウンロードされます。

PCを毎日使う人は、代わりに `スケジューラ登録.bat`(PC内で毎朝7時に収集)でも構いません。
クラウド連携をセットアップすると、こちらのPC内タスクは自動で解除されます。

## 注意

- 収益目安はコミュニティ推計(約$0.05/プレイ時間)に基づく概算です。
- 指標は前日確定分です(Epic側の仕様。7日分まで遡れます)。
- マップが表示されるには「公開＋Discoverable設定＋1日5人以上のプレイ」が必要です
  (自分のマップを追いたい場合もこの条件を満たせば検索対象になります)。
